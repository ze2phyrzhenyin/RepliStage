# 用户文档

## 1. 适用对象

本工具适合以下三类使用者：

- 演员
- 导演
- 课堂演示或排练观察者

## 2. 开始使用

### 第一步：进入首页

打开应用后，首页会显示剧目名称、角色列表和入口按钮。

### 第二步：选择角色

点击“选择角色，开始排练”，进入角色选择页。

你可以：

- 选择某个角色进行背词排练
- 选择“旁观者”查看全量台词

## 3. 排练页的三种模式

### 演出

这是默认模式，适合正式排练：

- 中间显示舞台
- 右侧显示简版台本
- 底部是播放控制条

### 台本

适合跟本和整体浏览：

- 左侧小舞台
- 右侧大台本
- 可快速定位当前事件

### 编排

适合临时调整当前场次：

- 可拖动角色修改走位
- 可编辑当前事件
- 可插入、删除、重排事件
- 修改只保存在当前浏览器的当前场次中

## 4. 演员如何背词

当你以某个角色进入时：

- 别人的台词会正常显示
- 你的台词默认不会直接展示
- 当前轮到你时，会显示“轮到你了”

如果需要提示，可以点击当前台词气泡显示这一句，再次点击可隐藏。

如果你只是想通读全文，也可以打开底部“全部台词”开关。

## 5. 播放控制

底部控制栏支持：

- 播放 / 暂停
- 上一句 / 下一句
- 重置到开头
- 拖动进度条跳转
- 切换播放速度
- 自动播放开关
- 全部台词开关

## 6. 键盘快捷键

排练页支持以下快捷键：

- `Space`
  播放 / 暂停
- `ArrowLeft`
  上一句
- `ArrowRight`
  下一句
- `1`
  切到“演出”
- `2`
  切到“台本”
- `3`
  切到“编排”

在编排模式下还支持：

- `Cmd/Ctrl + Z`
  撤销
- `Cmd/Ctrl + Shift + Z`
  重做
- `Cmd/Ctrl + Y`
  重做

## 7. 导演如何使用

### 轻量方式：排练页中的“编排”

适合排练时临时改：

- 拖动角色改走位
- 编辑台词和动作
- 重排事件顺序
- 重置回默认剧本

### 完整方式：导演模式

直接访问 `/director`：

- 管理多场次
- 编辑场次标题、副标题、说明
- 管理演员
- 调整门和椅子坐标
- 绘制路径走位
- 导入 JSON
- 导出 JSON
- 恢复示例剧本
- 保存到当前浏览器后，首页、选角页和排练页会一起使用当前剧本

## 8. 造型选择

首页和选角页都可以调整角色造型。

支持的造型包括：

- 默认剪影
- 已设计角色的不同肤色版本
- 动物造型

造型选择会保存在当前浏览器，下次打开仍会保留。

## 9. 常见问题

### 为什么我的台词不显示？

因为你进入的是该角色本人的排练模式，这是工具的设计目标。可以点击当前气泡临时显示，或打开“全部台词”。

### 为什么导演模式打不开？

通常是页面资源加载失败，或当前浏览器阻止了脚本运行，不再需要额外登录。

### 为什么我改完事件，刷新后没了？

如果你是在排练页“编排”模式下修改，数据只保存在当前浏览器，而且仅对应当前场次。清理浏览器数据或更换设备后不会同步。

### 为什么切到别的场次后看不到刚才的临时编排？

因为排练页“编排”模式的草稿是按场次分别保存的，这样可以避免多场次互相覆盖。

### 为什么造型会恢复默认？

通常是浏览器本地存储被清理，或者当前浏览器阻止了本地存储写入。
