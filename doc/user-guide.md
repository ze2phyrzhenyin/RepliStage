# 用户文档

## 1. 这个网站现在怎么用

首页只有两个入口：

- `开始排练`
- `导演模式`

无论进入哪一个，下一步都会先选择：

- 示例剧本
- 或导入 JSON

然后再选择第几节。

## 2. 排练模式

### 进入流程

1. 首页点 `开始排练`
2. 在 `/rehearsal/setup` 选择示例剧本或导入 JSON
3. 选择第几节
4. 进入角色选择页
5. 选择角色或旁观者
6. 进入排练页

### 你可以做什么

- 以某个角色身份背词
- 以旁观者身份查看完整台词
- 播放 / 暂停 / 快进
- 手动逐句推进
- 切换演出 / 台本 / 编排模式
- 临时调整当前场次事件

### 本人台词为什么默认不显示

如果你选择的是某个角色，系统会把该角色当前台词隐藏，帮助你练习记忆。

你仍然可以：

- 点击当前气泡临时显示
- 打开“全部台词”

## 3. 排练页三种模式

### 演出

- 适合正式排练
- 舞台为主
- 右侧台本为辅

### 台本

- 适合跟本
- 更强调事件列表和台词浏览

### 编排

- 适合临时调整当前场次
- 可拖角色
- 可插入、删除、编辑事件
- 只影响当前浏览器当前场次草稿

## 4. 导演模式

### 进入流程

1. 首页点 `导演模式`
2. 在 `/director/setup` 选择示例剧本或导入 JSON
3. 选择第几节
4. 进入导演工作区

### 你可以做什么

- 切换场次
- 编辑标题、副标题、场景说明
- 管理演员
- 编辑事件
- 绘制路径走位
- 拖动门和椅子
- 导入 JSON
- 导出 JSON
- 保存到浏览器
- 清除浏览器保存

### setup 页和导演页的区别

- setup 页负责选剧本和场次
- 导演页负责真正编辑

现在导演页顶部已经不再是示例剧本入口大厅。

## 5. 示例剧本

当前内置两部：

- `Cendrillon`
- `秦琼卖马`

你不需要先区分中法语，只需要选你要用哪部剧本。

## 6. 导入 JSON

排练模式和导演模式都可以导入 JSON。

导入时现在会先做：

- 结构校验
- 业务校验
- 结构预览

常见报错包括：

- 角色引用不存在
- 路径为空
- 缺少 `scene_end`
- 缺少门道具
- 重复 id

## 7. 导出 JSON

导出 JSON 不会清掉浏览器里的当前剧本。

也就是说：

- 保存到浏览器后，导出只是下载一份副本
- 刷新页面，本地浏览器内容仍在

## 8. 浏览器保存

当前剧本、排练草稿和造型都保存在浏览器本地。

优点：

- 刷新后仍在
- 不需要数据库

限制：

- 换设备不会同步
- 清浏览器数据会消失

导演模式里已经提供“清除浏览器保存”按钮。

## 9. 角色造型

选角页可以给角色切换造型。

当前支持：

- 角色原画
- 动物造型
- 剪影模式

造型保存在当前浏览器。

## 10. 常见问题

### 为什么切换示例剧本后内容变了

因为当前工作剧本已被切换到新的示例剧本。setup 流程选择的剧本就是当前工作剧本来源。

### 为什么排练页改完事件，别的设备看不到

因为排练页“编排”只保存在本地浏览器，而且按场次隔离。

### 为什么导演模式导入 JSON 后首页和排练也变了

因为它们共享同一个当前工作剧本。

### 为什么有时椅子不显示

因为现在椅子已经是显式道具。场景没有配置椅子时，就不会凭空出现。
