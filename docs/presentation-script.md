# RepliStage 功能展示稿

用途：向戏剧学院做功能展示  
建议时长：2 到 3 分钟

## 1. 开场

Bonjour à toutes et à tous. 今天我想向大家展示一个我使用 AI 辅助开发的话剧排练网站，名字叫 **RepliStage**。

它的目标很简单：帮助演员排练台词和走位，也帮助导演或老师更直观地组织一场戏的排练。

## 2. 首页

首页有两个入口：**开始排练** 和 **导演模式**。

今天展示时，我会使用法语剧本和法语界面，这样更接近法国戏剧学院的实际使用场景。

## 3. 选择剧本和场次

进入任意模式后，系统会先让我们选择剧本和场次。

目前网站内置了法语示例剧本 `Cendrillon`。除此之外，也可以导入自己的 JSON 剧本，用于不同课程、不同剧目或不同排练片段。

我的实际工作流程是：先用 AI 辅助把原始剧本整理成结构化数据，包括场次、角色、台词、动作、走位和道具事件，然后导入 RepliStage。这个能力以后也可以做成网站内置功能，只是当前版本先通过本地生成和导入来实现。

## 4. 排练模式

现在先看排练模式。

演员进入前可以选择自己的角色，也可以选择旁观者身份。选择角色时，还可以为角色更换不同服装或造型，方便区分人物，也可以尝试不同的舞台呈现效果。

选择角色后，系统会默认隐藏这个角色自己的台词。当轮到他说话时，页面只提示“轮到你了”，帮助演员练习记忆。如果需要，也可以临时显示当前句台词，或者查看全部台词。

排练页面可以播放、暂停、上一句、下一句，也可以拖动进度条跳到指定位置。舞台上会同步显示角色入场、退场、走位、动作、黑幕和道具变化。

也就是说，演员看到的不只是文字台本，而是台词和舞台调度结合在一起的排练过程。

## 5. 导演模式

第二部分是导演模式。

导演模式可以切换场次，编辑场景说明，管理角色，并修改每一个舞台事件。比如添加或修改台词、动作、入场、退场、走位和路径走位。

导演还可以直接管理舞台道具，比如添加椅子、门、床、水晶鞋、柜台、茶具等道具，调整它们的位置、标签、显示隐藏和图层顺序。

这样老师或导演在排练前，就可以先把文本、人物调度和舞台道具安排好；排练过程中如果发现问题，也可以直接调整。

## 6. 导入和导出

网站支持导入和导出 JSON。也就是说，一个剧本可以先在外部整理好，再导入系统排练；修改后的版本也可以导出，方便保存、分享或继续编辑。

当前版本主要是本地使用，内容保存在浏览器里，不需要账号和服务器。

## 7. 总结

总的来说，RepliStage 主要适合三类场景：

第一，演员背词和走位练习。  
第二，导演或老师进行场次编排。  
第三，戏剧课程中展示文本、角色和舞台调度之间的关系。

以上就是 RepliStage 的主要功能展示。Merci beaucoup.
